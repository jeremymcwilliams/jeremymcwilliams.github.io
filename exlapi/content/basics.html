API is short for Application Programming Interface.

An API is the interface implemented by an application which allows other applications to communicate with it.

It's like an access gateway to data from a service/website, provided you follow their rules & schema.

There are over 13,500 APIs on the web (6/30/15, Programmable Web)

REST is short for REpresentational State Transfer

General rule: Ask a question with a URL, get an answer back in some form (usually data, sometimes images and other things)
