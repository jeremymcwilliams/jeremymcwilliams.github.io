Create Analytics Report

Hi Alliance folks! This tutorial is designed for staff who are new to coding, and want to use the Alma APIs. In this tutorial you will:

  • Create and publish an Analytics report
  • Write code (PHP or Javascript) to use the Alma Analytics API to access the report
  • Create an output of the report as a web page

Follow the navigation links on the left from top to bottom to proceed through the tutorial.
