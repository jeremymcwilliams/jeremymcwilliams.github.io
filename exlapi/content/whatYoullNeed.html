What You'll Need

  • Requirements
  • Alma Bibs API Key
  • A user account for the Ex Libris Developer Network
  • An Alma Bibs API Key
  • A computer with a code editor (try TextWrangler for Mac, or Notepad ++ for Windows)
  • Set up your machine to run PHP
    • If you have a Mac, this isn't too hard to set up. Check out these results for more info.
    • For Windows, try QuickPHP.
  • A Worldcat API Key (this is optional, but needed to fully complete the Sample Application)
Here's how to get your Alma Bibs API Key:
  1. Visit the Ex Libris Developer Network and login with your credentials. This should redirect you to your Dashboard.
  2. Click "Applications" on the left, and then click the red "Add Application" button.
