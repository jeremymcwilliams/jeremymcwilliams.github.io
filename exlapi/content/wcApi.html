Use the WorldCat Search API LibraryLocations GetByISBN method

Purpose
find local libraries who own a title, given the ISBN. You'll use this to create the list of owning libraries for the app.
You need:
  • ISBN
  • Zip Code
  • Worldcat API Key
Request Format:
http://www.worldcat.org/webservices/catalog/content/libraries/isbn/{ISBN}?location={ZIP Code}&wskey={Wordcat API Key}
