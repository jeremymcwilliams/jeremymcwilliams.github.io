Welcome

This tutorial is designed for staff who are new to coding, and want to use the Alma APIs. In this tutorial you will:

  • Learn the basics of what an API is and how it works
  • Examine the following methods in the Bibs API
    • Retrieve Bib Record (requires MMS ID)
    • Scan In (requires barcode)
  • Write code in PHP or Javascript to use these methods
  • Create a sample app in PHP or Javascript, using the Scan In method and the Worldcat Search API

Follow the navigation links on the left from top to bottom to proceed through the tutorial.
