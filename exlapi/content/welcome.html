Welcome

Hi Alliance folks! This tutorial is designed for staff who are new to coding, and want to use the Alma APIs. In this tutorial you will:

  • Learn the basics of what an API is and how it works
  • Examine a few methods in the Bibs API and get an API key
  • Write code in javascript or PHP to use the Bibs "Retrieve Bib record" method
